# todolist
🚀 Features
✅ Add new tasks
📋 View all existing tasks
✏️ Update/Edit tasks
❌ Delete completed or unwanted tasks
🔄 Continuous menu-driven interface
🖥️ User-friendly command-line interaction



🛠️ Technologies Used
Programming Language: Python
Development Environment: Any Python-supported IDE (VS Code, PyCharm, IDLE, etc.)


📖 How It Works
The program displays a menu with different options.
Users can choose an action by entering the corresponding number.
Tasks are stored in a list during program execution.
Users can:
Add a task to the list.
View all current tasks.
Update an existing task.
Remove a task from the list.
The program continues running until the user chooses to exit.

Variables and Data Types
